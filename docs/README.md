# 前言
AI的发展前所未有的迅猛，国外国内的大厂正在抢占AI高地，甚至国内某些大厂正在梭哈AI，别问我怎么知道的，秘密！哈哈哈哈哈其实是皮一下，其实在网上一搜就有。

AI的基石是深度学习，而深度学习属于机器学习，所以系统性的学习机器学习还是很有必要的。

那么什么是机器学习呢？想象一下，如果你的电脑不仅能执行你告诉它的指令，还能自己**从经验中学习**，甚至逐渐变得比你更懂数据——这就是**机器学习**（****Machine Learning****, ****ML****）的魅力所在。简单来说，机器学习就是让计算机靠数据“自己长芝士”，从而预测未来、做决策、甚至帮你解决各种复杂问题。

现在，它已经渗透到我们的生活中：**银行用它评估信用风险，医院用它预测疾病风险，购物网站用它推荐你可能喜欢的商品，手机识别你拍的猫咪还是狗狗**……几乎无处不在。

我创建这个知识库的目标，就是带大家拆开**机器学习**的黑盒子，从最基础的理论、数学工具，到各种算法、工程实践，再到现实中的应用案例，逐层剖析机器学习的奥秘。无论你是新手小白，还是希望系统梳理知识的进阶者，都能在这里找到自己的方向。

在接下来的章节中，我会尽量让这些看似高深的概念变得易懂、可操作，甚至有趣，大家跟进别掉队哦！
